# EVBox Assignmnet Cypress

For completing the assignmnet, I have used cypress as testing framework. 
I have covered writing test cases for Create,edit, delete and comeplete from the list. I have also wriiten negative test cases scenario for each test cases.

## Getting Started:

1. Clone the source code
1. Pull the latest cypress code from master:
1. Install Cypress as a node package:

   npm i
1. Launch the Cypress runner: npx cypress open
